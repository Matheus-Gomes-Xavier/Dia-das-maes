💐 Projeto Especial: Dia das Mães

Este não é apenas um projeto de desenvolvimento web. É uma homenagem em forma de código dedicada às três mulheres da minha vida que me ensinaram o verdadeiro significado de força, resiliência e amor incondicional.  
📖 Sobre o Projeto

Desenvolvi esta página interativa como um presente digital para celebrar o Dia das Mães. O objetivo foi unir minhas habilidades técnicas em HTML, CSS e JavaScript para criar um espaço onde memórias e sentimentos pudessem ser visualizados e celebrados.  

A página conta com elementos interativos, uma galeria de fotos especial e uma trilha sonora que acompanha a navegação, tornando a experiência única para cada uma delas.  
🛠️ Tecnologias Utilizadas

Para construir essa homenagem, utilizei:

    HTML5: Estruturação semântica de toda a mensagem e galeria.  

    CSS3 (Animations & Flexbox): Criação dos corações flutuantes, efeitos de transição suave e um layout responsivo.  

    JavaScript: Lógica para controle da música, exibição de mensagens surpresa e o sistema de "Lightbox" para ampliar as fotos das memórias.  

💖 Funcionalidades Criadas

    Chuva de Corações: Uma animação contínua que simboliza o amor que sinto por elas.  

    Player de Música: Uma trilha sonora escolhida a dedo para emocionar durante a leitura.  

    Galeria de Memórias: Espaço dedicado para fotos com a vovó e com minhas mães, com efeito de zoom ao clicar.  

    Mensagem Surpresa: Um botão interativo que revela um agradecimento especial.  

💌 Uma Nota Pessoal

No código, deixei registrado o que sinto: cada momento ao lado de vocês é precioso. Este projeto é minha forma de agradecer por nunca terem desistido de mim e por tudo o que proporcionaram a mim e aos meus irmãos.  

    "Peço perdão pelos momentos de grosseria ou pelo quarto bagunçado, mas saibam que meu amor por vocês só cresce."
